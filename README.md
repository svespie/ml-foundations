# ML Foundations

A comprehensive collection of foundational concepts in mathematics, computer science, and machine learning. Organized as Jupyter notebooks with detailed explanations, visualizations, and code implementations.

## Directory Structure

```
ml-foundations/
├── math/                    # Mathematical foundations
│   ├── calculus/
│   ├── discrete/           # Set theory, combinatorics, graph theory, number theory, logic
│   ├── linear-algebra/
│   ├── probability/
│   └── statistics/
├── cs/                      # Computer science fundamentals
│   ├── algorithms/
│   ├── complexity-analysis/
│   └── data-structures/
├── ml/                      # Machine learning concepts
│   ├── supervised/         # Linear models, tree-based, neural networks
│   ├── unsupervised/       # Clustering, dimensionality reduction, autoencoders
│   └── reinforcement/
├── data/                    # Datasets
├── assets/                  # Images and diagrams
├── utils/                   # Reusable Python utilities
└── reference/              # Reference materials
```

## Setup

1. Clone the repository:
```bash
git clone <repository-url>
cd ml-foundations
```

2. Create and activate a virtual environment:
```bash
python -m venv venv
source venv/bin/activate  # On macOS/Linux
# venv\Scripts\activate   # On Windows
```

3. Install dependencies:
```bash
pip install -r requirements.txt
```

4. Launch Jupyter:
```bash
jupyter notebook
```
